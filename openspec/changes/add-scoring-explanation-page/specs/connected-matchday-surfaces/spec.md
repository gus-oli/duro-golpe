## ADDED Requirements

### Requirement: Scoring help SHALL be reachable from scoring-adjacent product surfaces
The authenticated product SHALL expose a clear path to the scoring reference from surfaces where users commonly need help interpreting points, standings, or special-market impact.

#### Scenario: User needs context while reviewing competition or score outcome
- **WHEN** an authenticated user is on a league, concluded match, or outright surface where scoring context matters
- **THEN** the interface MUST expose a visible path to the scoring reference page without forcing the user to return to the main menu first
